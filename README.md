//Artisan's Corner
A MERN Stack multi-vendor marketplace where artisans can showcase and sell handmade products while customers can browse, purchase, and manage orders.

//Features
- User Authentication (JWT)
- Seller Dashboard
- Product Management (CRUD)
- Shopping Cart
- Order Management
- Cloudinary Image Upload
- Responsive Design

//Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB Atlas
- Cloudinary
- Render
- Vercel

//Live Links
Frontend: https://artisans-corner-three.vercel.app/
Backend: https://artisanscorner.onrender.com
